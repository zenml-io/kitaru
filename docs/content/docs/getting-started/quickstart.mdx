This guide walks you through installing Kitaru and verifying that everything
works.

## Install

<Tabs items={["uv (recommended)", "pip"]}>
  <Tab value="uv (recommended)">
    ```bash
    uv add kitaru
    ```
  </Tab>
  <Tab value="pip">
    ```bash
    pip install kitaru
    ```
  </Tab>
</Tabs>

## Verify

Check that the CLI is available:

```bash
kitaru --version
```

You should see the installed version number (e.g. `0.1.0`).

Run `--help` to see available commands:

```bash
kitaru --help
```
