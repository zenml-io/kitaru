---
title: Installation
description: Install Kitaru with uv or pip
---

Kitaru requires **Python 3.11 or newer**.

You can verify your interpreter with:

```bash
python --version
```

<Tabs items={["uv (recommended)", "pip"]}>
  <Tab value="uv (recommended)">
    ```bash
    uv add kitaru
    ```
  </Tab>
  <Tab value="pip">
    ```bash
    pip install kitaru
    ```
  </Tab>
</Tabs>

## Optional extras

Install MCP server support:

```bash
uv add kitaru --extra mcp
# or: pip install "kitaru[mcp]"
```

Install PydanticAI adapter support:

```bash
uv add kitaru --extra pydantic-ai
# or: pip install "kitaru[pydantic-ai]"
```

## Verify Installation

```bash
kitaru --version
```

If you run `kitaru` or `kitaru-mcp` on an older Python version, Kitaru now
fails immediately with a clear error message instead of surfacing confusing
startup metadata noise.

```bash
kitaru --help
```
