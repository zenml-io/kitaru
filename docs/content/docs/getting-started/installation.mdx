---
title: Installation
description: Install Kitaru with uv or pip
---

Kitaru requires **Python 3.11 or newer**.

You can verify your interpreter with:

```bash
python --version
```

<Tabs items={["uv (recommended)", "pip"]}>
  <Tab value="uv (recommended)">
    ```bash
    uv add kitaru
    ```
  </Tab>
  <Tab value="pip">
    ```bash
    pip install kitaru
    ```
  </Tab>
</Tabs>

## Optional extras

Install MCP server support:

```bash
uv add kitaru --extra mcp
# or: pip install "kitaru[mcp]"
```

Install PydanticAI adapter support:

```bash
uv add kitaru --extra pydantic-ai
# or: pip install "kitaru[pydantic-ai]"
```

## Verify Installation

<Tabs items={["uv project", "pip environment"]}>
  <Tab value="uv project">
    ```bash
    uv run kitaru --version
    uv run kitaru --help
    ```
  </Tab>
  <Tab value="pip environment">
    ```bash
    kitaru --version
    kitaru --help
    ```
  </Tab>
</Tabs>

If you run `kitaru` or `kitaru-mcp` on an older Python version, Kitaru fails
immediately with a clear error message instead of surfacing confusing startup
metadata noise.

## Requirements

- Python 3.11 or later

## Initialize your project

After installing, initialize your project root so Kitaru knows where your
source code lives:

<Tabs items={["uv project", "pip environment"]}>
  <Tab value="uv project">
    ```bash
    uv run kitaru init
    ```
  </Tab>
  <Tab value="pip environment">
    ```bash
    kitaru init
    ```
  </Tab>
</Tabs>

This creates a `.kitaru/` directory in the current folder. It is used for
source root detection when packaging flows for remote execution. Add
`.kitaru/` to your `.gitignore`.
