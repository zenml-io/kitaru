---
title: Installation
description: Install Kitaru with uv or pip
---

<Tabs items={["uv (recommended)", "pip"]}>
  <Tab value="uv (recommended)">
    ```bash
    uv add kitaru
    ```
  </Tab>
  <Tab value="pip">
    ```bash
    pip install kitaru
    ```
  </Tab>
</Tabs>

## Verify Installation

```bash
kitaru --version
```

```bash
kitaru --help
```

## Requirements

- Python 3.12 or later
