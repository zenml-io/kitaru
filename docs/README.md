# Kitaru Documentation

This directory contains the [FumaDocs](https://fumadocs.dev/) documentation site
for Kitaru, deployed at [docs.kitaru.ai](https://docs.kitaru.ai).

## Prerequisites

- Node.js 22+ (pinned in `.node-version`)
- [pnpm](https://pnpm.io/) package manager

## Setup

```bash
cd docs
pnpm install
```

## Development

From the **repo root** (recommended):

```bash
just docs           # Start dev server at localhost:3000
just docs-build     # Full static build (output in docs/out/)
just docs-check     # Validate docs build
just generate-docs  # Generate CLI reference from Python source
```

Or from the `docs/` directory:

```bash
pnpm run dev        # Dev server with hot reload
pnpm run build      # Static export build
```

## Content Authoring

Documentation content lives in `content/docs/` as `.mdx` files (Markdown with
optional JSX components).

1. Create or edit an `.mdx` file in the appropriate directory
2. Add frontmatter with `title` and `description`
3. Update `meta.json` if adding a new page
4. Preview with `just docs`

See `CLAUDE.md` in this directory for detailed conventions and available
components.

## What's Generated vs Hand-Written

| Directory | Source | Committed? |
|---|---|---|
| `content/docs/getting-started/` | Hand-written | Yes |
| `content/docs/contributing/` | Hand-written | Yes |
| `content/docs/changelog/` | Hand-written | Yes |
| `content/docs/cli/` | Generated by `scripts/generate_cli_docs.py` | No (gitignored) |
| `content/docs/reference/python/` | Generated (deferred) | No (gitignored) |
